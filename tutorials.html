配置教程 Setup Tutorials

选择你的播放器，跟随步骤配置 IPTV 直播源。Windows / Mac / Linux 全覆盖。

PotPlayer 是 Windows 平台最强的本地播放器，原生支持 ASX 播放列表格式。 以下教程将教你如何导入世界杯频道列表并开始观看。

1

下载 TV.asx 播放列表

从 资源页 下载 TV.asx 文件。保存到桌面或任意目录。

2

打开 PotPlayer 播放列表面板

启动 PotPlayer，按 F2 键打开播放列表面板。或者在右侧列表面板空白处右键 → 添加文件。

3

导入 TV.asx 文件

点击"添加文件"，找到并选中下载的 TV.asx，点击打开。频道列表会自动加载到播放器中。

4

开始观看

双击任意频道名称即可开始播放。切换频道只需双击其他频道。

5

优化缓冲（可选）

如果播放卡顿，增大网络缓冲：右键 → 设置 → 播放 → 网络/缓冲 → 将网络缓冲改为 10000ms。

VLC Media Player 是跨平台开源播放器，支持 Windows / macOS / Linux。 推荐 Mac 和 Linux 用户使用 VLC 导入 .m3u8 播放列表。

1

下载播放列表

从 资源页 下载 playlist.m3u8 文件。

2

打开 VLC 播放列表

启动 VLC，按 Ctrl+L（或视图 → 播放列表）打开播放列表面板。

3

导入文件

将下载的 playlist.m3u8 文件直接拖入 VLC 播放列表面板，或点击"添加文件"按钮选择该文件。

4

开始观看

双击任意频道名称即可开始播放。如需切换频道，双击其他频道即可。

5

优化缓冲（可选）

如果播放卡顿，增大网络缓存：工具 → 偏好设置 → 左下角选"全部" → 输入/编解码器 → 网络缓存改为 10000ms。

自动更新脚本 可以从上游 IPTV 仓库拉取最新频道地址，确保播放列表长期可用。 支持 Windows 任务计划程序定时运行。

1

下载脚本

从 资源页 下载 Update-TVList.ps1 文件。

脚本会自动从 15bangjago/IPTV-Sport-New 仓库拉取最新频道列表，匹配世界杯转播频道，生成 TV.asx 文件。

2

手动运行

右键脚本 → "使用 PowerShell 运行"。或在终端执行：

powershell.exe -ExecutionPolicy Bypass -File ".\Update-TVList.ps1"

3

设置定时自动更新

使用 Windows 任务计划程序让脚本每天自动运行：

  1. 打开"任务计划程序"（Win+R → taskschd.msc）
  2. 创建基本任务 → 名称"IPTV频道更新"
  3. 触发器：每天 → 设置时间（如 09:00）
  4. 操作：启动程序 → powershell.exe
  5. 参数：-ExecutionPolicy Bypass -File "你的路径\Update-TVList.ps1"
4

脚本原理

脚本执行流程：从 GitHub 下载 1600+ 频道 → 匹配世界杯转播商 → 生成 TV.asx → 整个过程 3-5 秒。

常见问题 FAQ

频道无法加载 / Channel not loading

可能的原因及解决方案：

  1. 网络问题：检查网络连接。部分频道服务器在欧洲，中国用户可能需要代理。
  2. 频道已下线：IPTV 频道有时效性，下载最新播放列表即可。
  3. 防火墙阻挡：在防火墙中添加 PotPlayer/VLC 例外。
  4. 等待超时：首次连接可能需要 5-10 秒缓冲。
如何切换音频语言 / How to switch audio language

beIN Sports MAX 频道为阿拉伯语解说。如需其他语言：

  • 英语：TNT Sports (英国)、Eurosport
  • 法语：TF1、M6、Canal+ Sport (法国)
  • 意大利语：Rai 1、Rai 2
  • 德语：ARD
缓冲卡顿 / Buffering issues

改善缓冲的方法：

  1. PotPlayer：右键 → 选项 → 网络 → 缓冲区改为 10000ms
  2. VLC：工具 → 偏好设置 → 输入/编解码器 → 网络缓存改为 10000ms
  3. 使用有线网络代替 Wi-Fi
Mac / Linux 用户如何观看

推荐使用 VLC：

  • macOS：App Store 搜索 VLC 免费下载
  • Linux：sudo apt install vlc (Ubuntu/Debian)
  • 下载 playlist.m3u8，用 VLC 打开即可
播放列表多久更新一次

上游仓库不定期更新。建议每周检查一次资源页，或使用自动更新脚本设置每日定时更新。
