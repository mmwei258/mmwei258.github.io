配置教程 Setup Tutorials

选择你的播放器，跟随步骤配置 IPTV 直播源。Windows / Mac / Linux 全覆盖。

PotPlayer 是 Windows 平台最强的本地播放器，原生支持 ASX 播放列表格式。 以下教程将教你如何导入世界杯频道列表并开始观看。
PotPlayer is the best media player for Windows with native ASX playlist support.

1

下载 TV.asx 播放列表 Download the playlist

从 资源页 下载 TV.asx 文件（约 4KB）。保存到桌面或任意目录。

[截图：浏览器下载 TV.asx 的对话框]
2

打开 PotPlayer 播放列表面板 Open the playlist panel

启动 PotPlayer，在右侧播放列表面板空白处右键单击，选择 "添加文件"（或直接按 F2 键）。

[截图：PotPlayer 右键菜单 — 添加文件]
3

导入 TV.asx 文件 Import the ASX file

在弹出的文件选择对话框中，找到并选中你下载的 TV.asx 文件，点击"打开"。

[截图：文件选择对话框，选中 TV.asx]
4

频道列表加载成功 Channels loaded

播放列表面板会出现 29 个频道，按国家分组显示。可以看到频道名称和分组信息。

[截图：PotPlayer 播放列表已加载 29 个频道，显示国家分组]
5

双击播放，开始观看！ Double-click to watch

在列表中双击任意频道名称即可开始播放。等待 3-5 秒缓冲后即可观看。

替代方法：按 Ctrl+U 打开 URL 输入框，直接粘贴 M3U8 流地址。

[截图：PotPlayer 正在播放世界杯比赛画面]

VLC Media Player 是跨平台开源播放器，支持 Windows / macOS / Linux。 原生支持 M3U8 播放列表和网络流，适合导入全球 IPTV 频道列表。
VLC is a cross-platform open-source player with native M3U8 support.

1

下载 playlist.m3u8 Download the playlist

从 资源页 下载 playlist.m3u8（约 518KB，含 1622 个全球频道）或 TV.asx。

[截图：浏览器下载 playlist.m3u8 文件]
2

用 VLC 打开播放列表 Open with VLC

启动 VLC。菜单栏选择 媒体 → 打开文件（或按 Ctrl+O），选择 playlist.m3u8。VLC 会自动解析并加载所有频道。

[截图：VLC 打开文件对话框]
3

方法二：网络流直接播放 Network stream method

如果你只想看一个频道，可以 媒体 → 打开网络串流（Ctrl+N），粘贴单个 M3U8 地址即可。

[截图：VLC 网络串流对话框，粘贴 URL]
4

浏览和播放 Browse and play

频道会加载到播放列表中。使用 Ctrl+L 打开播放列表面板，双击频道即可观看。VLC 支持频道搜索和收藏。

[截图：VLC 播放列表已加载，正在播放频道]

自动更新脚本 可以定时从 IPTV 源仓库拉取最新频道列表，自动更新你的 TV.asx 文件。适合想保持频道列表始终最新的用户。
Auto-update script keeps your playlist in sync with the latest channel list.

1

下载 Update-TVList.ps1 脚本 Download the update script

PowerShell 脚本会从以下三个源拉取最新频道数据：

  • Free-TV/IPTV — 主播放列表
  • iptv-org/iptv — 全球频道
  • 15bangjago/IPTV-Sport-New — 世界杯专用频道

脚本会自动筛选世界杯相关的转播频道（TF1、M6、beIN Sports、TNT Sports、Sky Sports、ESPN、DAZN 等），生成 TV.asx 文件。

2

运行脚本 Run the script

右键点击 Update-TVList.ps1，选择 "使用 PowerShell 运行"。或者在 PowerShell 中执行：

powershell.exe -ExecutionPolicy Bypass -File ".\Update-TVList.ps1"

运行后会在当前目录生成/更新 TV.asx 文件。

3

设置定时任务（可选） Schedule auto-update

Windows 任务计划程序可以每天自动运行脚本：

  1. 打开"任务计划程序"（Win+R → taskschd.msc）
  2. 创建基本任务 → 名称"IPTV频道更新"
  3. 触发器：每天 → 设置时间（如 09:00）
  4. 操作：启动程序 → powershell.exe
  5. 参数：-ExecutionPolicy Bypass -File "你的路径\Update-TVList.ps1"

常见问题 FAQ

频道无法加载 / Channel not loading

可能的原因及解决方案：

  1. 网络问题：检查网络连接。部分频道服务器在欧洲，中国用户可能需要代理。
  2. 频道已下线：IPTV 频道有时效性，部分频道可能已失效。下载最新播放列表即可。
  3. 防火墙阻挡：Windows 防火墙或杀毒软件可能阻止 PotPlayer/VLC 联网，请在防火墙中添加例外。
  4. 等待超时：首次连接可能需要 5-10 秒缓冲，请耐心等待。
如何切换音频语言 / How to switch audio language

beIN Sports MAX 频道为阿拉伯语解说，目前不支持音频轨道切换。如需其他语言解说，请选择对应国家的频道：

  • 英语：TNT Sports (英国)、Eurosport
  • 法语：TF1、M6、Canal+ Sport (法国)
  • 意大利语：Rai 1、Rai 2
  • 德语：ARD
缓冲卡顿 / Buffering issues

改善缓冲的几种方法：

  1. PotPlayer：右键 → 选项 → 网络 → 增加缓冲区大小（默认 5000ms → 10000ms）
  2. VLC：工具 → 偏好设置 → 输入/编解码器 → 网络缓存改为 10000ms
  3. 使用有线网络代替 Wi-Fi
  4. 避开高峰期（比赛开始前15分钟流量最大）
Mac / Linux 用户如何观看 / How to watch on Mac or Linux

推荐使用 VLC Media Player（跨平台支持）：

  • macOS：App Store 搜索 VLC 免费下载
  • Linux：sudo apt install vlc (Ubuntu/Debian)
  • 下载 playlist.m3u8，用 VLC 打开即可
播放列表多久更新一次 / How often is the playlist updated

播放列表源（15bangjago/IPTV-Sport-New）不定期更新。我们建议：

  • 每周检查一次资源页面是否有新版本
  • 使用自动更新脚本（Update-TVList.ps1）设置每日定时更新
  • 世界杯期间频道通常比较稳定，不易失效
