VLC Media Player 是跨平台开源播放器，支持 Windows / macOS / Linux。 推荐 Mac 和 Linux 用户使用 VLC 导入 .m3u8 播放列表。
VLC is a free cross-platform media player. Recommended for Mac and Linux users.

1

下载播放列表 Download playlist

从 资源页 下载 playlist.m3u8 文件（全频道列表，约 500KB）。

2

打开 VLC 播放列表 Open in VLC

启动 VLC，按 Ctrl+L（或视图 → 播放列表）打开播放列表面板。

3

导入文件 Import file

将下载的 playlist.m3u8 文件直接拖入 VLC 播放列表面板，或点击"添加文件"按钮选择该文件。

4

开始观看 Start watching

在播放列表中双击任意频道名称即可开始播放。如需切换频道，双击其他频道即可。

5

优化缓冲（可选） Optimize buffering

如果播放卡顿，增大网络缓存：

  1. VLC 菜单 → 工具 → 偏好设置（Preferences）
  2. 左下角"显示设置"改为"全部"（All）
  3. 左侧找到"输入/编解码器"（Input / Codecs）
  4. 右侧找到"网络缓存"（Network caching），改为 10000ms
  5. 保存后重启 VLC

自动更新脚本 可以从上游 IPTV 仓库拉取最新频道地址，确保播放列表长期可用。 支持 Windows 任务计划程序定时运行。
Auto-update script fetches the latest channels from upstream IPTV repositories.

1

下载脚本 Download the script

从 资源页 下载 Update-TVList.ps1 文件。

脚本会自动从 15bangjago/IPTV-Sport-New 仓库拉取最新频道列表，匹配世界杯转播频道，生成 TV.asx 文件。

2

手动运行 Run manually

在脚本所在目录，右键点击 Update-TVList.ps1 → 选择"使用 PowerShell 运行"。或者在终端执行：

powershell.exe -ExecutionPolicy Bypass -File ".\Update-TVList.ps1"

运行后会在当前目录生成 TV.asx 文件，覆盖旧版本。

3

设置定时自动更新 Schedule auto-update

使用 Windows 任务计划程序让脚本每天自动运行：

  1. 打开"任务计划程序"（按 Win+R，输入 taskschd.msc）
  2. 右侧点击"创建基本任务"
  3. 名称：IPTV频道更新
  4. 触发器：每天 → 设置时间（如 09:00）
  5. 操作：启动程序
  6. 程序或脚本：powershell.exe
  7. 参数：-ExecutionPolicy Bypass -File "C:\你的路径\Update-TVList.ps1"
  8. 点击完成，任务将每天自动更新播放列表
4

脚本原理 How it works

脚本执行流程：

  1. 从 GitHub 下载 15bangjago/IPTV-Sport-New 的频道列表（约 1600+ 频道）
  2. 按关键词匹配世界杯转播商频道（TF1、TNT Sports、beIN Sports、Alkass 等）
  3. 生成 TV.asx 播放列表文件
  4. 整个过程约 3-5 秒完成
