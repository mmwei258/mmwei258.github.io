关于 FREE IPTV WORLD

让每个人都能免费观看世界杯直播。

项目故事 Our Story

FREE IPTV WORLD 是一个非商业、社区驱动的开源项目，致力于为全球用户提供免费的体育赛事直播观看方案。 项目最初源于一个简单的想法：2026 年世界杯期间，许多球迷在社交媒体上反复问同一个问题——"哪里能免费看直播？"

我们调研后发现，互联网上其实存在大量公开的 IPTV 频道资源（如 GitHub 上的 iptv-org、15bangjago/IPTV-Sport-New 等仓库）， 但普通用户面临三个障碍：不知道这些资源的存在、不知道哪些频道真正可用、不知道怎么导入播放器。

于是 FREE IPTV WORLD 诞生了。我们做了三件事：聚合公开资源、验证连通性、教程化操作步骤。 从 1,622 个候选频道中精选出覆盖 11 个国家、6 种语言的 29 个世界杯转播频道，用 ffprobe 逐一验证。

我们承诺：播放列表永远免费。永不收费订阅。所有频道来源均为互联网公开资源。

工作原理 How It Works

📡 数据来源

频道列表来自 GitHub 公开 IPTV 仓库（15bangjago/IPTV-Sport-New、iptv-org/iptv、Free-TV/IPTV）。定期同步上游更新。

🔍 频道验证

使用 ffprobe 对每个频道进行连通性测试，验证是否为有效的 MPEG-TS 实时流。

🎯 精选过滤

从上千个频道中筛选拥有世界杯版权的转播频道。按国家、语言分类。

📦 多格式输出

提供 ASX（PotPlayer）、M3U8（VLC/通用）、CSV（数据分析）三种格式。

🔄 自动更新

PowerShell 自动更新脚本，可设置每日定时从上游拉取最新频道列表。

🌍 双语支持

全站中英双语，服务国内和国际用户。教程、频道列表、FAQ 均提供中英文版本。

常见问题 FAQ

这个项目收费吗？ / Is this project free?

完全免费。 播放列表下载、教程、频道列表全部免费。我们通过 affiliate 推广链接和用户打赏维持运营，但绝不会对核心内容收费。

你们提供流媒体服务吗？ / Do you provide streaming?

不提供。 我们不运营任何流媒体服务器，不托管视频内容。所有频道链接指向公开的第三方 IPTV 源。

频道突然不能看了怎么办？ / What if a channel stops working?

在 GitHub Issues 报告失效频道（附频道名 + 日期），或下载最新播放列表。

可以贡献频道或提建议吗？ / Can I contribute?

非常欢迎！ 通过 GitHub Issues 提交新频道、报告失效、改进文档、分享项目给更多人。

频道有地区限制怎么办？ / What about geo-restrictions?

使用 VPN 连接频道所在国家；选择 beIN Sports MAX 系列（中东源无地区限制）；或选择同一国家其他免费频道。
